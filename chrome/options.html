Unofficial MMC to Onshape Importer Settings

First Time Usage:

  1. Set up API Keys (one time only)
    1. Open Onshape in a new window, leave this window open so you can follow instructions
    2. Log into Onshape
    3. Click on your avatar in the top right, then click "My account"
    4. Click Developer in the left pane
    5. Click API Keys, (click "Accept" if you get a popup)
    6. Click Create New API Key
    7. Give it a useful name like McMasterHelper
    8. Check "Application can read your documents" AND "Application can write to your documents"
    9. Click Create API Key
    10. Copy the access AND secret keys from the popup to the boxes below
    11. Click Save
  2. Open OnShape Part Document where you want your McMaster parts to go (it is auto-detected)
  3. Open McMaster, find part that you want to add
  4. Choose STEP file type
  5. Click the extension icon in your browser toolbar to open the side panel (one time).
  6. The "🚀 Send to Onshape" button will turn blue once you select the STEP format — just click it!
  7. In 1-5 seconds, the file should show up in your Onshape Document as a new tab on the bottom

Regular Usage:

  1. Open Onshape Document
  2. Open McMaster
  3. Left click on the "Unofficial MMC to Onshape Importer" icon to bring up the sidebar (pinning the extension might help)
  4. Find a part, select STEP
  5. Click the "🚀 Send to Onshape" button in the sidebar
  6. In 1-5 seconds, the file should show up in your Onshape Document as a new tab on the bottom

"Advanced" Onshape Usage:

  1. The extension auto-detects open Onshape Documents
  2. If you have multiple Onshape Documents open, you should use the "Target Onshape Document URL" below.
  3. The extension will push to the "Target Onshape Document URL" even if it is not open
